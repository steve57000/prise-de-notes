
# Langages de Programmation depuis 1950

Ce fichier décrit les principaux langages de programmation créés depuis les années 1950, en présentant leur contexte historique, leur but, et des exemples d'utilisation.

---

## 1. **Fortran** (1957)
### Description
Fortran (FORmula TRANslation) est considéré comme le premier langage de programmation de haut niveau, conçu pour des calculs scientifiques et techniques.

### Exemple d'utilisation
```fortran
PROGRAM HelloWorld
  PRINT *, "Hello, World!"
END PROGRAM HelloWorld
```

---

## 2. **LISP** (1958)
### Description
LISP (LISt Processing) a été inventé pour la recherche en intelligence artificielle. Ce langage est le premier à introduire les concepts de récursivité et de traitement de listes.

### Exemple d'utilisation
```lisp
(defun hello-world ()
  (print "Hello, World!"))
(hello-world)
```


---
## 3. **ALGOL** (1958)
### Description
ALGOL (Algorithmic Language) a influencé la conception de nombreux langages modernes, comme C et Pascal, en introduisant des concepts comme la structure de blocs.

### Exemple d'utilisation
```algol
begin
  print("Hello, World!")
end
```


---

## 4. **COBOL** (1959)
### Description
COBOL (COmmon Business-Oriented Language) est conçu pour les applications de gestion et d'entreprise, avec un accent sur la simplicité syntaxique pour favoriser la lecture et la maintenance du code.

### Exemple d'utilisation
```cobol
IDENTIFICATION DIVISION.
PROGRAM-ID. HelloWorld.
PROCEDURE DIVISION.
    DISPLAY 'Hello, World!'.
    STOP RUN.
```

---

## 5. **BASIC** (1964)
### Description
BASIC (Beginner's All-purpose Symbolic Instruction Code) a été conçu pour être simple et accessible aux débutants en programmation, notamment dans le cadre éducatif.

### Exemple d'utilisation
```basic
PRINT "Hello, World!"
```

---

## 6. **Pascal** (1970)
### Description
Pascal a été créé pour enseigner les concepts de la programmation structurée, particulièrement utilisé dans le monde académique.

### Exemple d'utilisation
```pascal
program HelloWorld;
begin
  writeln('Hello, World!');
end.
```


---

## 7. **Prolog** (1972)
### Description
Prolog est un langage logique principalement utilisé en intelligence artificielle et en traitement des données.

### Exemple d'utilisation
```prolog
hello :- write('Hello, World!').
```

---

## 8. **Smalltalk** (1972)
### Description
Smalltalk est un langage orienté objet, influent dans la création de nombreux autres langages de programmation orientés objet.

### Exemple d'utilisation
```smalltalk
Transcript show: 'Hello, World!'.
```


---

## 9. **INTERCAL** (1972)
### Description
INTERCAL est un langage humoristique conçu pour être difficile à lire et contraire aux conventions des langages habituels. Il est plus souvent étudié pour son caractère atypique que pour une utilisation pratique.

### Exemple d'utilisation
```intercal
DO ,1 <- #13
PLEASE DO ,1 SUB #1 <- #238
DO ,1 SUB #2 <- #112
DO ,1 SUB #3 <- #94
PLEASE READ OUT ,1
```

---

## 10. **C** (1972)
### Description
Développé par Dennis Ritchie, le langage C a été créé pour le développement du système d'exploitation UNIX. Il est à la base de nombreux autres langages modernes.

### Exemple d'utilisation
```c
#include <stdio.h>
int main() {
   printf("Hello, World!");
   return 0;
}
```

---

## 11. **Python** (1991)
### Description
Python est connu pour sa syntaxe simple et lisible, ainsi que pour son utilisation dans divers domaines, de l'analyse de données au développement web.

### Exemple d'utilisation
```python
print("Hello, World!")
```

---
## 12. **Brainfuck** (1993)
### Description
Brainfuck est un langage de programmation minimaliste conçu pour être difficile à comprendre et à écrire. Il utilise uniquement huit commandes et fonctionne avec un tableau de cellules mémoire.

### Exemple d'utilisation
```brainfuck
++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++.>.+++.------.--------.>+.>.
```


---

## 13. **Java** (1995)
### Description
Java est un langage orienté objet conçu pour être portable et indépendant de la plateforme, utilisé dans les applications d'entreprise, les appareils mobiles, et plus encore.

### Exemple d'utilisation
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```


---

## 14. **JavaScript** (1995)
### Description
JavaScript est le langage de script du web. Il est principalement utilisé pour rendre les pages web interactives et dynamiques.

### Exemple d'utilisation
```javascript
console.log("Hello, World!");
```

---

## 15. **Ruby** (1995)
### Description
Ruby est un langage de programmation dynamique et orienté objet, apprécié pour sa simplicité et utilisé principalement dans le développement web.

### Exemple d'utilisation
```ruby
puts "Hello, World!"
```

---

## 16. **PHP** (1995)
### Description
PHP est un langage de script côté serveur conçu pour le développement web, intégré dans de nombreux sites et applications en ligne.

### Exemple d'utilisation
```php
<?php
echo "Hello, World!";
?>
```



---


## 17. **C#** (2000)
### Description
Développé par Microsoft, C# est un langage orienté objet destiné au développement d'applications sur la plateforme .NET.

### Exemple d'utilisation
```csharp
using System;

class HelloWorld {
    static void Main() {
        Console.WriteLine("Hello, World!");
    }
}
```

---

## 18. **Shakespeare** (2001)
### Description
Shakespeare est un langage de programmation dans lequel le code ressemble à une pièce de théâtre, avec des personnages et des dialogues représentant les instructions.

### Exemple d'utilisation
_(Les programmes ressemblent à du texte théâtral)_

## 19. **Ook!** (2001)
### Description
Ook! est un langage de programmation basé sur le Brainfuck, mais chaque commande est remplacée par des combinaisons de "Ook.". Il est souvent utilisé comme une forme humoristique.

### Exemple d'utilisation
```ook
Ook. Ook? Ook! Ook! Ook. Ook?
```

---


## 20. **Whitespace** (2002)
### Description
Whitespace est un langage de programmation qui ignore tous les caractères sauf les espaces, les tabulations, et les sauts de ligne. Ce langage est conçu pour être invisible dans le code.

### Exemple d'utilisation
_(Le code de Whitespace n'est visible que par des espaces, tabulations et sauts de ligne)_


---

## 21. **Piet** (2002)
### Description
Piet est un langage de programmation visuel où les programmes sont écrits sous forme d'images. La séquence et les couleurs des pixels définissent les instructions. 

### Exemple d'utilisation
_(Un exemple visuel est nécessaire)_

---


## 22. **Chef** (2002)
### Description
Chef est un langage conçu pour que les programmes ressemblent à des recettes de cuisine. Les variables sont des ingrédients, et les instructions sont des étapes de cuisine.

### Exemple d'utilisation
_(Chef est rédigé comme une recette)_

---

## 23. **LOLCODE** (2007)
### Description
LOLCODE est un langage humoristique qui imite la syntaxe des "lolcats", avec une grammaire simpliste et des commandes en anglais mal orthographié.

### Exemple d'utilisation
```lolcode
HAI 1.2
CAN HAS STDIO?
VISIBLE "HAI WORLD!"
KTHXBYE
```

---

## 24. **Go** (2009)
### Description
Go, développé par Google, est conçu pour la performance et la simplicité, particulièrement adapté pour les applications serveur et les microservices.

### Exemple d'utilisation
```go
package main
import "fmt"
func main() {
    fmt.Println("Hello, World!")
}
```

---

## 25. **Rust** (2010)
### Description
Rust est un langage de programmation conçu pour être sécurisé et performant, particulièrement dans le développement de systèmes.

### Exemple d'utilisation
```rust
fn main() {
    println!("Hello, World!");
}
```

---

### Description
Développé par Apple, Swift est un langage moderne conçu pour remplacer Objective-C dans le développement d'applications iOS et macOS.

### Exemple d'utilisation
```swift
print("Hello, World!")
```
