
---

## 1. margin
Contrôle l'espace à l'extérieur d'un élément.

```html
<div style="margin: 10px;">
  Avec marge.
</div>
```


## 2. padding

Contrôle l'espace à l'intérieur d'un élément, entre le contenu et la bordure.
```html
<div style="padding: 10px;">
  Avec padding.
</div>
```

**Shorthand pour margin/padding :**
```css
margin: 10px 20px 30px 40px; /* haut, droite, bas, gauche */
padding: 5px 10px; /* haut/bas, gauche/droite */
```
