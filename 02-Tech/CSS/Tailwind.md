# Cours Tailwind CSS

## Qu'est-ce que Tailwind CSS ?
Tailwind CSS est un framework CSS utilitaire qui permet de construire des interfaces rapidement et facilement en utilisant des classes prédéfinies.

## Installation
Pour utiliser Tailwind CSS, vous devez l'installer via npm ou inclure le CDN dans votre fichier HTML.

### Installation via npm
```bash
npm install -D tailwindcss
```
### Inclusion via CDN
```html
<link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">
```

## Utilisation de Tailwind CSS
### Exemple de structure de base
```html
<div class="bg-blue-500 text-white p-4 rounded">
    <h1 class="text-2xl">Bienvenue dans Tailwind CSS</h1>
    <p>Ceci est un paragraphe d'exemple.</p>
</div>
```

### Classes utilitaires
- **Couleurs** : `bg-red-500`, `text-green-700`
- **Espacement** : `p-4`, `m-2`
- **Typographie** : `font-bold`, `text-xl`

## Conclusion
Tailwind CSS permet une personnalisation facile et rapide grâce à ses classes utilitaires. Explorez la [documentation officielle](https://tailwindcss.com/docs) pour en savoir plus.
