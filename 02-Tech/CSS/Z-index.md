
La propriété `z-index` contrôle l'ordre d'empilement des éléments. Plus la valeur est élevée, plus l'élément est au-dessus.  
### Exemple : 

```html
<div style="position: absolute; z-index: 10;">Cet élément est au-dessus.</div>
```

### Note :

- Fonctionne uniquement sur des éléments positionnés (`relative`, `absolute`, `fixed`).
