# Cours Bootstrap

## Qu'est-ce que Bootstrap ?
Bootstrap est un framework CSS populaire qui facilite la conception de sites Web réactifs et modernes grâce à un ensemble de composants préconçus.

## Installation
Vous pouvez inclure Bootstrap via CDN ou en l'installant via npm.

### Inclusion via CDN
```html
<link href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css" rel="stylesheet">
```

### Installation via npm
```bash
npm install bootstrap
```

## Utilisation de Bootstrap
### Exemple de structure de base
```html
<div class="container">
    <h1 class="text-center">Bienvenue dans Bootstrap</h1>
    <button class="btn btn-primary">Cliquez ici</button>
</div>
```

### Composants Bootstrap
- **Boutons** : `btn`, `btn-primary`, `btn-secondary`
- **Grille** : `row`, `col`, `col-md-4`
- **Alertes** : `alert`, `alert-success`, `alert-danger`

## Conclusion
Bootstrap simplifie la création de designs modernes et réactifs. Consultez la [documentation officielle](https://getbootstrap.com/docs/4.5/getting-started/introduction/) pour plus de détails.
