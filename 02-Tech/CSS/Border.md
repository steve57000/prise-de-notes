
La propriété `border` permet de définir des bordures autour des éléments.
## 1. border-width 

Définit la largeur de la bordure.
```html 
<div style="border-width: 5px;"> Bordure épaisse. </div>
```


## 2. border-style

Définit le style de la bordure (solid, dotted, dashed, etc.).

```html
 <div style="border-style: dashed;">   Bordure pointillée. </div>
```

## 3.border-color

Définit la couleur de la bordure.

```html
<div style="border-color: blue;">Bordure bleue.</div>
```

## 4. border-radius

Permet de créer des coins arrondis.

```html
<div style="border-radius: 10px;">Bordure arrondie.</div>
```
