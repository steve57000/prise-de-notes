
# Culture Informatique

## Le matériel (hardware)
Le **hardware** désigne tous les composants physiques d'un ordinateur. Cela inclut :
- Le processeur (CPU) : le cerveau de l'ordinateur, qui exécute les instructions des programmes.
- La mémoire vive (RAM) : mémoire temporaire qui stocke les données utilisées par le processeur pour exécuter des tâches en cours.
- Le disque dur (HDD/SSD) : unité de stockage de longue durée où les données sont enregistrées.
- Les périphériques d'entrée/sortie : clavier, souris, écran, imprimante, etc.
- La carte graphique (GPU) : spécialisée dans le traitement des données visuelles pour l'affichage.

## Le logiciel (software)
Le **software** fait référence aux programmes et systèmes qui fonctionnent sur le matériel. Il peut être classé en plusieurs catégories principales :
#### Les systèmes d'exploitation
Un système d'exploitation (OS) est le logiciel principal qui gère le matériel et les autres logiciels. Les systèmes d'exploitation les plus connus sont :
- **Windows** : développé par Microsoft, utilisé sur une grande majorité des PC.
- **macOS** : système d'exploitation des ordinateurs Apple.
- **Linux** : une famille de systèmes d'exploitation open-source, souvent utilisée par les développeurs et pour les serveurs.
- **Android** et **iOS** : systèmes d'exploitation mobiles pour smartphones et tablettes.

#### Les logiciels
Les logiciels peuvent être classés en deux catégories :
- **Logiciels applicatifs** : destinés à l'utilisateur final pour effectuer des tâches spécifiques, comme les navigateurs web, les suites bureautiques (ex. : Microsoft Office), les éditeurs de texte, etc.
- **Logiciels systèmes** : ils permettent au matériel de fonctionner avec le logiciel, comme les pilotes de périphériques et les utilitaires.

## L'algorithme
Un **algorithme** est une série d'instructions définies et ordonnées, permettant de résoudre un problème ou d'accomplir une tâche précise. Chaque algorithme doit être :
- Fini : il doit se terminer après un nombre d'étapes limité.
- Déterministe : pour les mêmes entrées, il doit produire toujours les mêmes sorties.
- Correct : il doit donner la solution au problème posé.

#### Premier algorithme (-3000 avant J-C)
L'un des premiers exemples d'algorithme remonte à l'époque des Sumériens (~3000 avant J-C), avec l'invention des techniques de calcul pour la gestion des ressources agricoles et des impôts. Ces méthodes incluaient des règles pour l'addition, la soustraction et la gestion des transactions économiques.
