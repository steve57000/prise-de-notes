## ==Définition==
_____________________________________________________________

#### Protocole

- Instruction précise et détaillée mentionnant toutes les opérations à effectuer dans un certain ordre ainsi que les principes fondamentaux à respecter pour exécuter une opération, réaliser une expérience.

- En Informatique  
Ensemble des règles qui permettent à un utilisateur de se connecter sur un réseau ou à diverses parties de ce réseau de communiquer entre elles. (Ce mot est fondamentalement synonyme de procédure). Son emploi est cependant réservé aux cas présentant un certain degré de complexité.

#### Couche

- Architecture  
Pièce de bois posée horizontalement à terre, sur laquelle s'appuient des étais


## ==Communication & Protocoles==
_______________________________________________________

Pour communiquer :

- Une source
- Une destination
- Un “lien” entre la source et la destination
- Un encodage
- Un protocole : Ensemble de règles

#### Protocoles

Une communication implique plusieurs type d’éléments :

- physique
	- “Objet” 
		- câbles, ondes
- Adressage:
	- MAC
- logique / virtuelle
    - Routage
	    - IP
    - Contenu du message
	    - TCP
    - Spécificités liées au contexte d’utilisation : HTTP / SSH
