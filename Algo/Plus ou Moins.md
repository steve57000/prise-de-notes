# Jeu Python : C'est plus, C'est moins

## Description
Ce jeu consiste à deviner un nombre aléatoire généré par l'ordinateur entre 1 et 100. Le joueur dispose de 10 tentatives pour trouver le bon nombre, en étant guidé par des indications "C'est plus" ou "C'est moins".

## Fonctionnalités
- Génération d'un nombre aléatoire entre 1 et 100.
- Le joueur a 10 tentatives pour deviner le nombre.
- Utilisation de la bibliothèque `colorama` pour colorer le texte selon les tentatives restantes.
- Affichage des messages personnalisés en fonction des tentatives restantes (nombre de victoires et défaites).
- Possibilité de rejouer après une partie.

## Prérequis
- Python 3.x
- Bibliothèque `colorama` pour la gestion des couleurs dans le terminal.

### Installation des dépendances
Pour installer les bibliothèques nécessaires, tu as deux options :

1. **Si le fichier `requirements.txt` est présent** :  
   Exécute la commande suivante pour installer toutes les dépendances :
   ```bash
   pip install -r requirements.txt
   ```

2.  **Sinon :**
   ```bash
   pip install colorama
   ```
