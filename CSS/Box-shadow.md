
La propriété `box-shadow` permet de créer des ombres autour des éléments. 

### Syntaxe : 

```css 
box-shadow: h-offset v-offset blur-radius spread-radius color;
```


**Exemple :**
```html
<div style="box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);">Ombre portée.</div>
```

- **h-offset** : Décalage horizontal.
- **v-offset** : Décalage vertical.
- **blur-radius** : Rayon de flou.
- **spread-radius** : Expansion de l'ombre.
- **color** : Couleur de l'ombre.
