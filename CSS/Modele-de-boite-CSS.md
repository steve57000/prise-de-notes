
# Le modèle de boîte en CSS

## Qu'est-ce que le modèle de boîte ?
Chaque élément d'une page HTML est considéré comme une boîte en CSS.

## Les propriétés principales
- **Padding** : espace entre le contenu de l'élément et sa bordure.
    ```css
    div { padding: 20px; }
    ```
- **Border** : la bordure autour de l'élément.
    ```css
    div { border: 1px solid black; }
    ```
- **Margin** : espace entre la bordure de l'élément et les autres éléments.
    ```css
    div { margin: 15px; }
    ```

## Exemple visuel

```html
<div style="margin: 20px; padding: 10px; border: 2px solid blue;">
    Contenu ici
</div>
```
