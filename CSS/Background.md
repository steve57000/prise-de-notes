

---

## 1. background-color
Définit la couleur de fond d'un élément.

```html
<div style="background-color: lightblue;">Fond bleu clair.</div>
```


## 2. background-image

Définit une image de fond.

```html
<div style="background-image: url('image.jpg');">
  Fond avec image.
</div>
```

## 3. background-position

Définit la position de l'image de fond.

```html
<div style="background-position: center;">Image centrée.</div>
```

## 4. background-size

Définit la taille de l'image de fond.

```html
<div style="background-size: cover;">
  Image de fond étirée pour couvrir.
</div>
```
