
---


La propriété `overflow` détermine ce qui se passe lorsque le contenu d'un élément dépasse la taille du conteneur.

## 1. overflow
Valeurs possibles : `visible`, `hidden`, `scroll`, `auto`.

```html
<div style="overflow: hidden;">Contenu masqué si trop grand.</div>
```


## 2. overflow-x et overflow-y

Contrôlent le débordement horizontal et vertical.

```html
<div style="overflow-x: scroll;">
  Défilement horizontal.
</div>
```
