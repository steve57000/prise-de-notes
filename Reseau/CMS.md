**Def:**
CMS => CONTENT MANAGEMENT SYSTEME

## ==Gestion de contenue numérique :==

- Création
- modification

## ==Gestion de contenu dans l'entreprise :==

-  gestion d'ensemble des contenus non structurés
	*exemples : ensemble d'informations d'un dossier client des documents / courrier liés*

## ==Système de gestion de contenu Sur Internet== 

- **gestion de contenu web**
	-mise en page
	-navigation
	
- **Offres :**
	-téléchargeable & installable
	-utilisable en ligne
	
- **Types :**
	-Offline: pré rendu et déploiement
	-Online : rendu en direct
	-Hybride : pré rendu en langage dynamique
	
## ==CMS Web versus Site Web statique==

###### CMS
	Les +
		- Gestion de rôles (Admin / Editeur / Auteur / Lecteur)
		- Modèles de rendus
		- Edition WYSIWYG
		- Peu cher
	Les -
		- Génération de la page à la demande
		- Utilise le même serveu pour l'admin
		- Implémentation
		- Veille de sécurité

###### Site Web Statique
	Les +
		- rapide
		- Pas de langage serveur nécessaire
	Les -
		- Mise à jour par l'administarteur
		- Pas de contenu dynamiquement ajoutable
		- Connaissance en HTML / CSS / Javascript nécessaire

###### Autres avantages des CMS
	- Fonctions embarquées pour la gestion
		 - de média
		 - des langues
		 - des versions
		 - des standars (HTML / TAG)
		 - du SEO
		 - de thèmes
	- Plugin
